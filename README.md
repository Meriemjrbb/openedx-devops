# Open edX DevOps Project

## Contexte

Projet réalisé dans le cadre du stage DevOps RIF.

Infrastructure :
- OpenStack
- Ubuntu 22.04
- Docker
- Docker Compose
- Open edX (Tutor)

VM :
- 2 vCPU
- 6 Go RAM
- 20 Go disque

## Objectifs

- Déployer Open edX
- Automatiser avec Terraform
- Automatiser avec Ansible
- Documenter l'infrastructure
- Mettre en place une démarche DevOps
